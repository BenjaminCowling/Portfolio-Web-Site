About Me

I am a technically minded and proactive professional with a lifelong passion for technology. With a BSc (Hons) in Multimedia Technology and a strong foundation in Music Production, my career spans over two decades across the IT, live audio, and high-volume hospitality sectors.

Throughout my 3+ years managing head office operations and 10+ years facilitating venue audio, I consistently acted as the "unofficial" IT lead — migrating company data to cloud systems, upgrading physical network infrastructure, and troubleshooting hardware faults. I thrive in high-pressure environments, using my background in 5-star customer service and live sound engineering to de-escalate stressful situations and solve problems on the fly.

Following a recent redundancy, I seized the opportunity to formalise my technical expertise by self-funding my CompTIA A+ certification and launching a freelance IT consultancy. I pride myself on my communication skills and my ability to translate complex technical solutions into business-friendly language. This portfolio serves as a living record of my work, showcasing my ability to bridge the gap between technical infrastructure, audio-visual production, and day-to-day business operations.

How This Site Is Built

Self-hosted from scratch — no third-party platforms, no page builders.

Entirely self-hosted on a custom Raspberry Pi 5 — every layer from hardware to deployment is configured and maintained by hand. Rapid development assisted by Antigravity AI for pair-programming and deployment automation; all creative direction and technical decisions are my own.

Hardware

  • Raspberry Pi 5 — custom Debian server
  • PCIe Gen 3 NVMe storage via HAT
  • Active PWM cooling system
  • Dual ZFS pools — application data & mass media separated

Web Server

  • Apache2 with name-based virtual hosts
  • Dedicated vhost for portfolio.musicat.me
  • Custom error and access logging
  • Hand-coded HTML, CSS & JavaScript — no frameworks
  • AI-assisted development via Antigravity (agentic coding assistant)

Networking & Security

  • Cloudflare Tunnel — zero-trust HTTPS exposure
  • No open ports — all traffic routed via encrypted tunnel
  • Tailscale mesh VPN for remote SSH administration
  • Pi-hole for network-wide DNS sinkholing

Media & Storage

  • BackBlaze B2 cloud bucket for media assets
  • Audio, video & images served via B2 public URLs
  • B2 CLI for headless uploads and management
  • ZFS for local data integrity and redundancy

Deployment & Version Control

  • GitHub repository for source code
  • SCP-based deployment over Tailscale VPN
  • SSH key authentication — passwordless access
  • Fully headless workflow — no GUI required

Architecture Flow

Local Dev → GitHub → SCP → Pi / Apache → Cloudflare → You
Media Files → B2 CLI → BackBlaze B2 → Browser
